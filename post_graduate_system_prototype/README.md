# 🎓 Postgraduate Management System
## 📌 Overview

This project is a web-based system designed to support the management of postgraduate academic processes.

It provides role-based dashboards for different users to interact with and monitor academic activities.

## 🎯 Purpose

- Improve visibility of academic processes  
- Support structured workflow management  
- Provide a centralized system for user interaction  

## 🏗️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js  
- Database: MongoDB Atlas  

## ⚙️ Setup

```bash
git clone <repo-url>
cd project-folder
npm install
npm run dev
```

## 🏆 Note

This project was developed as part of a hackathon and is intended for demonstration purposes.
